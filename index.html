Cost modifier:

By default spells cost 50gp per level to scribe. This value modifies that, so a cost modifier of .5 would mean 25gp per level, while a modifier of 2 would mean 100 per level.

Time modifier:

By default spells tale 1 hour per level to scribe. This field allows you to modify that.

Flat time:

This field will set a fixed time (in minutes) to scribe spells regardless of level. If used this overrides the time modifier field.
